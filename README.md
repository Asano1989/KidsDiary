# アプリ名称：「おもいでつむぎ」または「きみのあしあと」

## サービス概要
「子どもの微笑ましい発言や面白かった行動を、記録して残しておきたい」

「子どもの成長について家族で共有したい」

「これまでの子どもの様子や思い出を振り返りたい」

保護者・養育者のこんな思いに応えるための、育児日記アプリです


## このサービスへの思い・作りたい理由
「子どもの面白い発言や、印象に残った行動はたくさんあるはずなのに、

しばらくすると記憶が薄れてしまったり、いつどこで起きたことだったのか忘れてしまうことがあるんだよね」

きっかけは育児中の妹の言葉でした。私も、伯母として妹の子の様子を近くで見てきたため、妹の言うことには深く共感しました。

子どもの発言や行動は、大人にとっては思いもよらぬことの連続です。

無邪気に繰り出される言葉で核心を突いてきたり、子どもらしい認識や思考で大人の笑いを誘ったりと、実に刺激的です。

印象に残る言葉や行動は何となく覚えているものですが、それが「何歳のどの時期のできごとだったのか」「いつ頃までその言動を繰り返していたのか」などは育児の忙しさや月日の流れの中で霞み、悲しいことにやがて記憶から失われていってしまいます。

何歳のいつ頃に、どんなことが好きで、どんなことに関心を持っていて、どういった言葉を発し、どういう行動を繰り返していたのか――。

大切な子どもの成長について忘れないために、いつか家族で振り返って思い出を語れるようになるために、子どもの日々の言動を記録しておく。

そのために作られたのが、この「おもいでつむぎ」です。


## ユーザー層について
ターゲットユーザーは以下の通りです。
 - 子どもの言動をメインに、気軽に育児日記をつけたい人
 - 家族で育児日記を共有したい人
 - 子どもの成長を振り返って、思い出に浸る機会がほしい人


## サービスの利用イメージ
1. ユーザー登録をして、家族（グループ）を設定する
1. 子どもたちの微笑ましい、面白い言動を目の当たりにする
1. ユーザーは手元のスマートフォンでそれを記録する
1. 日記（記録）は家族（グループ）で共有でき、いつでもどこでも子どもの記録にアクセス
1. 検索機能やカレンダーから記事を閲覧し、子どもの成長を振り返ることができる


## ユーザーの獲得について
 - 身近にいる子育て中の保護者・養育者に勧める
 - さらに、彼・彼女らに口コミで周囲にお勧めしてもらう
 - XをはじめとするSNSにアカウントを作成し、機能や魅力を発信する


## サービスの差別化ポイント・推しポイント

### 子どもの言動と成長にフォーカス
子どもの成長を記録するためのアプリはありますが、身長や体重の記録など、詳細な母子手帳のような役割を果たすものとなっています。

また、育児日記はデジタル端末でよりも紙の日記に書くものという考えがメジャーであるように見受けられます。

家族やグループ単位で交換日記を作成できるサービスはいくつかありますが、特定の人物やテーマにフォーカスしたものは稀です。

「おもいでつむぎ」において、主体となるのは子どもたちであり、**その言動や成長を記録し、家族で共有して振り返る機能に特化**した日記アプリです。


### 写真や動画ではなく、文章で思い出を残す
今は誰もがスマートフォンを持ち、写真や動画を気軽に撮れる時代です。お手軽でありながら、そのときの様子がしっかり残る写真や動画は、子どもの成長と言動を克明に捉えます。

子どもたちの写真と動画のアルバム機能に特化したアプリケーションは、子育て世帯とその身内に欠かせない存在となっています。

しかしながら、後々まで印象に残る瞬間を常に捉えるのは難しいのも事実です。「あの瞬間をカメラで残していたなら！」と後悔することもしばしば。

そんなとき、**あえて「文章」で子どもの言葉や行動を書き残し、忘れられない思い出となるように記録できる**のが「おもいでつむぎ」です。

### 類似サービスとの比較
 - [育児記録アプリ ぴよログ](https://www.piyolog.com/)
 - [家族ダイアリー](https://kazokudiary.nyanta.jp/)
 - [家族アルバム みてね](https://mitene.us/)

| アプリ名            | 内容                                                         | 強み                                                                             | おもいでつむぎの特色                                                             | 
| ------------------- | ------------------------------------------------------------ | -------------------------------------------------------------------------------- | -------------------------------------------------------------------------------- | 
| ぴよログ            | リアルタイムで共有できる育児記録アプリ                       | 子どもに関する記録（身長や体重、予防接種、日記）を逐一保存し振り返ることができる | 数字の記録よりも、子どもの微笑ましい**発言**や面白い**行動**にフォーカス               | 
| 家族ダイアリー      | 日々のできごとを写真付きで投稿してシェアできる交換日記アプリ | 家族や友達や恋人などで日記を共有できる                                           | メンバー同士の交換日記ではなく、**子どもを主体にした**日記                           | 
| 家族アルバム みてね | 子どもの写真や動画をかんたん共有、整理できるアプリ           | 写真と動画がメインのアルバムアプリであり、家族やメンバーがコメントできる         | **文章での記録**がメインの日記アプリで、コメントではなく**絵文字でリアクション**ができる | 


## 機能一覧

### MVPリリース時
   - ユーザーアカウント機能
     - ユーザーアカウント登録
     - ユーザー情報編集
   - 家族（グループ）機能
     - 家族（グループ）を作成（作成時点で、家族（グループ）管理ユーザーとなる）
     - 家族（グループ）のメンバー一覧を表示
     - 家族（グループ）のメンバーを追加（家族（グループ）管理ユーザーのみ）
     - 家族（グループ）のメンバーを削除（家族（グループ）管理ユーザーのみ）
     - 家族（グループ）の「子ども」を追加（家族（グループ）管理ユーザーのみ）
     - 家族（グループ）の「子ども」を編集・削除（家族（グループ）管理ユーザーのみ）
   - 日記（記録）機能
     - 日記（記録）を投稿
     - 日記（記録）を編集
     - 日記（記録）の表示
     - 日記（記録）を削除
   - 日記のカテゴリ分け機能
     1. 対象となる子どもによるカテゴリ分け
     1. 絵文字によるカテゴリ分け
     - カテゴリごとに日記（記録）一覧を表示
   - カレンダー表示機能
     - 日記（記録）をカレンダーに表示
     - 1日ごとの日記（記録）をまとめて表示
   - 絵文字による日記（記録）へのリアクション機能

### 本リリース時
   - Googleログイン機能
   - 日記（記録）に画像を添付して投稿できる機能
   - 日記（記録）のタグ付け機能
   - タグごとに日記（記録）一覧を表示する機能
   - 日記（記録）の検索機能
   - 日記（記録）の投稿時に「お題」を投げる機能
   - 「X年前の今日」の日記（記録）を表示する機能


## 使用する技術スタック
 - バックエンド：Ruby on Rails
 - フロントエンド：React + TypeScript
 - データベース：PostgreSQL、Supabase（ユーザー認証）
 - デプロイ先：DBはNeon、APはfly.io
 - 使用予定のライブラリ（Gem）：gemoji、simple_calendar、Action Mailer、ransack、act_as_taggrable_on、kaminari、rubocop、Tailwind CSS
